1.개요

Dakku Calendar(이하 '앱')는 Windows 데스크톱 애플리케이션으로, 일정 관리, 할 일, 일기, 스티커 꾸미기, 루틴 체크 등을 제공합니다. 본 방침은 앱이 어떤 정보를 수집·이용·저장하는지 설명합니다.

2.수집하는 정보와 저장 위치

3.구글 사용자 데이터 이용 목적

앱은 Google Calendar API를 사용하여 다음 기능을 제공합니다.

  • 사용자 전용 "Dakku Calendar" 캘린더 생성
  • 앱에서 작성한 일정을 구글 캘린더에 생성·수정·삭제
  • 구글 캘린더의 기존 일정을 앱 화면에 불러와 표시

수집된 구글 캘린더 데이터는 위 기능 제공 목적 외에는 사용되지 않으며, 광고, 프로파일링, 제3자 판매 등에 이용되지 않습니다.

4.제3자 제공 여부

앱은 수집한 어떠한 개인정보도 제3자에게 판매하거나 제공하지 않습니다. 구글 캘린더 데이터는 사용자 본인의 구글 계정과 사용자 기기 사이에서만 동기화됩니다.

5.데이터 보관 및 삭제

로컬에 저장된 데이터는 앱을 삭제하거나 설정에서 초기화하면 함께 삭제됩니다. 구글 계정 연동은 앱 내 '연동 해제' 버튼 또는 구글 계정 권한 설정에서 언제든지 해제할 수 있으며, 해제 시 앱이 보유한 인증 토큰은 즉시 폐기됩니다.

6.보안

구글 인증 토큰은 사용자 기기 로컬 저장소에만 보관되며, 앱 서버는 별도로 존재하지 않습니다(모든 처리는 사용자 기기에서 이루어짐).

7.아동의 개인정보

본 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동으로부터 고의로 개인정보를 수집하지 않습니다.

1.Overview

Dakku Calendar ("the App") is a Windows desktop application offering schedule management, to-do lists, a diary, decorative stickers, and routine tracking. This policy explains what information the App collects, uses, and stores.

2.Information Collected & Storage

FeatureData CollectedStorage Location
To-dos / Routines / Diary / Stickers / BannerText and images entered by the userLocal storage on the user's device only (never transmitted to any external server)
Google Calendar IntegrationGoogle account auth token; calendar event data (title, time, description)OAuth token stored locally on device. Event data syncs only between the user's own Google account and the App
Excel schedule importSchedule data from a user-selected Excel fileProcessed locally, then sent to Google Calendar only with user consent
Activity (usage time) trackingActive program / window title (optional feature, user can start/stop)Local storage only, never transmitted

3.Use of Google User Data

The App uses the Google Calendar API to:

  • Create a dedicated "Dakku Calendar" calendar for the user
  • Create, update, and delete events on the user's Google Calendar based on in-app entries
  • Display the user's existing Google Calendar events within the App

Google Calendar data is used solely for the purposes above and is never used for advertising, profiling, or sold to third parties.

4.Third-Party Sharing

The App does not sell or share any collected personal data with third parties. Google Calendar data is synced only between the user's own Google account and their own device.

5.Data Retention & Deletion

Locally stored data is removed when the user uninstalls the App or resets it from Settings. Google account access can be revoked at any time via the in-app "Disconnect" button or through Google Account Permissions; upon revocation, any stored auth token is immediately discarded.

6.Security

Google auth tokens are stored only in local storage on the user's device. The App does not operate a backend server — all processing happens on-device.

7.Children's Privacy

This App is not directed at children under 14, and does not knowingly collect personal information from children.

Any changes to this policy will be posted on this page.
